Create and run a react app in windows:
install node, npm

Create new folder, cd into it in powershell and run:
	npm create vite@latest chat-frontend -- --template react 
	cd into the created project folder (chat-frontend) or open the project in cursor
		To run :
			npm install
			npm run dev
		To build:
			npm run build

To run the app with a server based on package.json file:
            "scripts": {
            "dev": "vite",
            "build": "tsc -b && vite build",
            "lint": "eslint .",
            "preview": "vite preview"
        },
    cd dist
    npm run preview
    npm install -g serv
    python -m http.server 9000
    serv
